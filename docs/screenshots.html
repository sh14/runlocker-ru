Скриншоты утилиты Unlocker

Посмотрите скриншоты программы Unlocker для разблокировки процеесов и файлов.

Скриншоты программы:

  1. Кликните на файл правой клавишей мыши

  2. Если файл заблокирован появится диалоговое окно

  3. В появившемся окне выбирите действие, которое надо произвести с файлом(переименовать, удалить, переместить)
  4. Затем нажмите кнопку "Разблокировать все" или выбирите нужный процеес и нажмите "Разблокировать"
Данный сайт не является официальным сайтом программы Unlocker. Все права защищены и принадлежат разработчику - Empty Loop. Данный ресурс не несет ответственности за скачанные файлы, так как не влияет на их содержимое.
